# 📚 Library Management System (Python + SQLite)

A **console-based Library Management System** built using **Python** with **SQLite database integration**.  
This project demonstrates modular programming, database handling, and basic CRUD operations in a real-world use case.

---

## 🚀 Features

- 📖 View all available books  
- 📥 Borrow books from the library  
- 📤 Return issued books  
- 🎁 Donate new books  
- 🗂️ Track issued books with student names  
- 💾 Persistent data storage using SQLite  
- 🧩 Modular code structure for maintainability  

---

## 🛠️ Tech Stack

- **Programming Language:** Python  
- **Database:** SQLite  
- **Concepts Used:**
  - Object-Oriented Programming (OOP)
  - Modular Programming
  - Database Handling (SQLite)
  - Exception Handling
  - SQL CRUD Operations
